Si es la primera vez que abres esto (clonaste el repo):
1. Crear el venv: python -m venv [nombre1]
2. Activar el venv: [nombre1]\Scripts\activate
3. Instalar dependencias: pip install -r requirements.txt
   (NO usar django-admin startproject, el proyecto ya existe)

Cada vez que inicies el IDE o reinicies tu computadora:
[nombreProyecto]\Scripts\activate

Para correr el server y ver la página:
python manage.py runserver

Si creas tablas nuevas en models.py:
python manage.py makemigrations
python manage.py migrate

Si nomas haces cambios menores (como los __str__), solo guarda el archivo y refresca la página

Para crear superusuarios (cuentas de admin):
python manage.py createsuperuser

Si instalas una librería nueva (ej. algo con pip install):
pip freeze > requirements.txt
y haz commit de ese archivo para que el equipo lo tenga actualizado

Si es la primera vez que trabajas en la parte de FullCalendar (o jalas la rama vista-calendario):
1. Instalar Node.js (LTS) desde nodejs.org si no lo tienes
   Verificar con: node -v / npm -v
2. Instalar las librerías de FullCalendar:
   npm install @fullcalendar/core @fullcalendar/resource @fullcalendar/resource-timeline @fullcalendar/interaction @fullcalendar/daygrid
   npm install fullcalendar-scheduler
3. Copiar el archivo combinado a la carpeta de estáticos (PowerShell):
   copy node_modules\fullcalendar-scheduler\index.global.min.js reservaciones\static\reservaciones\fullcalendar\scheduler.js

Nota: node_modules/ NO se sube a GitHub, cada quien lo genera localmente con npm install
